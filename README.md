This Repo is the codebase for the prototype of Energy Consumption Simulator for the Buildings based on various paramenters like
wall material , Roof type, Window layout, facade size etc.

In this Prototype we have used Random Forest after trying various regression techniques. since the data was very well structured as it was derived from simulator platforms like Energy Plus (I dont know Exact name of platform but is similar to energy plus).

This codebase has **Verify_accuracy** named Python script which gives R^2 value for the Model.
This is just a demo dashboard.
